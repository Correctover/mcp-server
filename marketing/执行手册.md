## 阶段一：MCP 平台上架（5个平台）

### 1.1 Official MCP Registry（最权威）

**方式A：mcp-publisher CLI（推荐）**
```bash
curl -L "https://github.com/modelcontextprotocol/registry/releases/latest/download/mcp-publisher_$(uname -s | tr '[:upper:]' '[:lower:]')_$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/').tar.gz" | tar xz mcp-publisher
sudo mv mcp-publisher /usr/local/bin/
cd /path/to/mcp-server
mcp-publisher login github
mcp-publisher publish
```

**方式B：PR 到官方 registry**
- Fork: https://github.com/modelcontextprotocol/registry
- 在 servers/ 目录下添加 correctover/ 目录
- 复制 registry/server.json 内容
- 提交 PR

**验证：**
```bash
curl "https://registry.modelcontextprotocol.io/v0.1/servers?search=correctover"
```

**审核时间：** 2-5个工作日

### 1.2 Smithery（最大流量）

**方式A：CLI**
```bash
npm install -g @smithery/cli
smithery mcp publish "https://github.com/Correctover/mcp-server" -n Correctover/correctover
```

1. 访问 https://smithery.ai
