Why HTTP 200 is not enough.

One line. Two deps. Full protection.

一行代码，两个依赖，全维度保护。

No proxy. No infra changes. No data leaving your process.

不需要代理，不需要改基础设施，数据不离开进程。

# Before: unprotected failover — any HTTP 200 means "success"
response = openai.chat.completions.create(model="gpt-4o", messages=[...])

# After: verified failover with 6-dim validation
from correctover import SelfHealingEngine

engine = SelfHealingEngine(
    providers=["openai", "anthropic", "deepseek"],
    contract="strict"
)
response = engine.run("gpt-4o", messages=[...])
# openai fails → failover to anthropic → 6-dim validation
# Contract violated? → try deepseek → validated → safe return
