# Correctover Reference Implementation

> **Status**: Private — Internal Reference Implementation of the Correctover Governance Protocol

This repository contains the internal reference implementation of the Correctover Conformance Standard. It is not publicly distributed.

For research inquiries regarding the protocol's architecture, please refer to:
- **Published Standards**: [Correctover/standards](https://github.com/Correctover/standards) (CC BY 4.0)
- **Zenodo DOI**: 10.5281/zenodo.21191484

The protocol specification is publicly available. The reference implementation is maintained exclusively by the Correctover Standards Team to ensure empirical integrity and prevent adversarial adaptation of conformance tests.
